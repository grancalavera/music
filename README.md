# Music Toolbox

A kitchensink-style desktop application for macOS that provides various utilities for music production workflows.

## Status

**In Development** - Currently setting up the project infrastructure.

## Tech Stack

- **Tauri** - Desktop framework
- **Solid** - Frontend framework
- **Tailwind CSS** - Styling
- **Kobalte** - Accessible UI components
- **SQLite** - Local storage

## Features

### Current
- [ ] Ableton File Reader - Parse `.als` files and display project summaries

### Planned
- Roland SP-404 / P-6 Drum Pack Creator
- Chord Progression Tool
